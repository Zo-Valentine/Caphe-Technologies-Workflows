## üîç How to Use

1. **Browse by Category**: Navigate to the category folder that matches your industry or use case
2. **Explore Subcategories**: Each category contains specialized subcategories
3. **Review Workflows**: Each workflow includes:
   - Workflow JSON file (`.json`)
   - Documentation (`README.md`)
   - Metadata file (`metadata.json`)
   - Setup instructions

4. **Import to n8n**:
   - Copy the JSON file content
   - Open your n8n instance
   - Go to Workflows ‚Üí Import
   - Paste the JSON content
   - Configure your credentials and settings

## üìù Workflow Metadata

Each workflow includes metadata with the following information:
- **Name & Description**: Clear explanation of functionality
- **Difficulty Level**: Beginner, Intermediate, or Advanced
- **Required Integrations**: External services needed
- **Tags**: Searchable keywords
- **Use Case**: Real-world application example
- **Requirements**: Prerequisites and setup needs
